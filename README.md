# Motorola Sales Dashboard
An interactive Power BI dashboard designed to visualize Motorola’s sales performance across regions and time periods.  
This project demonstrates advanced data modeling, KPI tracking, and dashboard design for business decision-making.

## Objectives
- Provide a clear view of sales KPIs across regions.
- Enable drill-down analysis for monthly and quarterly performance.
- Support management in identifying growth opportunities.

## Tools & Technologies
- Power BI
- Excel

## Project Files
- **power bi dashboard.pbix**: Interactive Power BI dashboard.
- **Mobile Sales Data.xlsx**: Dataset used for analysis.

## Methodology
1. **Data Preparation**: Cleaned and structured raw sales data for consistency.
2. **Data Modeling**: Built relationships between tables to enable dynamic reporting.
3. **Dashboard Design**: Created interactive visuals for KPIs, trends, and regional breakdowns.
4. **Insights Generation**: Highlighted top-performing regions and identified declining trends.

## Key Contributions
- Developed a professional dashboard with drill-down capabilities.
- Automated KPI tracking for monthly and quarterly reporting.
- Delivered actionable insights for regional managers.

## Impact
- Highlighted top-performing regions contributing 35% of total sales.
- Improved decision-making speed by providing real-time insights.
- Enhanced visibility into sales trends and regional performance.

## Learning Outcomes
- Strengthened Power BI dashboard design and visualization skills.
- Gained experience in KPI analysis and trend reporting.
- Improved ability to communicate insights through interactive dashboards.
